# six

A simple Python program that demonstrates various mathematical properties of the number 6.

## Features

- Shows that 6 is a perfect number (the sum of its proper divisors equals the number itself)
- Calculates factorial, square, and cube of 6
- Lists all divisors of 6
- Demonstrates mathematical properties

## Usage

```bash
python3 six.py
```

or

```bash
./six.py
```

## What makes 6 special?

6 is the first perfect number! A perfect number is a positive integer that is equal to the sum of its proper positive divisors (divisors excluding the number itself).

For 6: proper divisors are 1, 2, and 3, and 1 + 2 + 3 = 6.
